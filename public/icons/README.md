# PWA Icons

Os ícones **icon-192.png** e **icon-512.png** são gerados a partir do vetor `public/icon.svg` (sem perda de resolução).

Para regenerar após editar o SVG:

```bash
npm run icons
```

- **icon.svg** — ícone vetorial (coroa dourada em fundo escuro); usado como favicon no browser.
- **icon-192.png** / **icon-512.png** — exportados em alta resolução para PWA e “Adicionar ao ecrã inicial”.
